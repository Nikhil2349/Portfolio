EV Market Analysis

Project Description

This project focuses on analyzing the growth and adoption trends of Electric Vehicles (EVs) across different Indian states. Through in-depth Exploratory Data Analysis (EDA), visualizations, and time series forecasting, the study uncovers regional adoption patterns, evaluates infrastructure readiness, and predicts future EV growth.

It also maps EV manufacturers across states to understand market dynamics. The analysis aids in identifying key regions for EV expansion and provides actionable insights for policymakers and manufacturers.

Project Details

  • Model: ARIMA
  • Skills: Python Pandas Matplotlib Seaborn Plotly NumPy Statsmodels

Insights

Regional disparities in EV adoption were revealed, with certain states leading. Growth is trending upwards, especially in recent years. Charging station distribution correlates with EV adoption. The ARIMA model provided reliable future projections for EV adoption and market expansion.

Twitter Sentiment Analysis

Project Description

This project performs sentiment analysis on real-time Twitter data related to the Indian Premier League (IPL) to classify public opinion into Positive, Negative, or Neutral categories. By combining Natural Language Processing (NLP) with machine learning models, the analysis reveals fan perceptions of IPL teams, players, and events. The study also includes geographic and temporal sentiment trends, offering a comprehensive view of audience engagement during the tournament.

Project Details

  • Model: Logistic Regression , Random Forest , Navie Bayes
  • Skills: Python Twikit Matplotlib Text Blob NLP Tokenization

Insights

Random Forest achieved the highest overall accuracy.Logistic Regression delivered the best cross-validation performance.WordClouds and visualizations highlighted frequent IPL-related topics and emotions.Sentiment varied geographically and temporally, indicating peak engagement during matches.Tweets with positive sentiment received more likes on average.

Credit Card Fraud Detection

Project Description

This project aims to build a robust and accurate machine learning solution to detect fraudulent credit card transactions. By analyzing transaction patterns and classifying transactions as either fraudulent (1) or normal (0), the project addresses real-world financial fraud prevention challenges.

It uses anonymized data and explores key behavioral patterns to improve fraud detection accuracy.

Project Details

  • Model: Logistic Regression , Random Forest , XG Boost
  • Skills: Python Pandas Matplotlib Seaborn Confusion Matrix XG Boost

Insights

XGBoost significantly outperformed Logistic Regression and Random Forest in fraud detection precision. High recall ensured minimal missed fraudulent transactions. Visualizations clearly conveyed fraud patterns and model performance metrics.
